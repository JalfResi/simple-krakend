# Simple Gateway API 

### Requirements

Docker [Install](https://docs.docker.com/install/)

### Installation

```shell
docker-compose up -d
```

### Endpoints

Access: http://localhost:8080/

#### / 

Show message

#### /nick/{username} 

Show profiles of Github/Bitbucket
